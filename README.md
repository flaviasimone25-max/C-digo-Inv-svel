# Código Invisível

Landing page de vendas (VSL + página de conversão) construída com TanStack Start e React.

## Deploy na Vercel

1. Importe o repositório no [Vercel](https://vercel.com)
2. Framework: **TanStack Start** (detectado automaticamente)
3. Build: `npm run build`
4. Install: `npm install`
5. (Opcional) Defina `VITE_SITE_URL` com o domínio final

## Desenvolvimento local

```bash
npm install
npm run dev
```

## VSL (Wistia)

O vídeo está configurado em `src/lib/vsl-config.ts`.  
Para embed funcionar em qualquer domínio, desbloqueie o link de compartilhamento no painel Wistia.
